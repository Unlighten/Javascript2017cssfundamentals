CSS Notes

The purpose of CSS is

  • CSS can be included within html elements using the style attribute
  • CSS can be used with the style html element
  • CSS is altering colors, fonts, backgrounds, etc.
  • CSS can implement some effects and animations

CSS Statements

Four main components include the following:

  1. Declaration: the statement as a whole with curly braces.
  2. Selector: the name of the part you want to style.
  3. Property: properties to edit(e.q., width, height, background, etc.)
  4. Value: numbers, colors, urls - values that we assign.
